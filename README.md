# 📝 Notes Manager

A simple command-line Notes Manager built using Python and File Handling.

## 📌 About the Project

This application allows users to create, view, search, and delete notes. All notes are stored in a text file, so they remain saved even after the program is closed.

## ✨ Features

* ➕ Add a new note
* 📖 View all saved notes
* 🔍 Search notes
* 🗑️ Delete all notes
* 💾 Save notes permanently using a text file
* 🚪 Exit the application

## 🛠️ Technologies Used

* Python 3
* File Handling (`open()`)
* Text Files (`notes.txt`)

## 📂 Project Structure

```text
main.py
notes.txt
README.md
```

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```
